nvim keymap cheatsheet

Leader = Space. Text objects: i = inside, a = around. Prefix with n/l for next/last (mini.ai). Wait after a prefix → mini.clue popup.

Quickfix / location list

Quickfix is global; location list is per-window. Most commands have l* twins.

[q ]qPrev / next qf entry (mini.bracketed)
\q / \QToggle qf / loclist window
:cdo {cmd}Run {cmd} on each qf entry (line-by-line)
:cfdo {cmd}Run {cmd} per qf file  (faster for %s/…)
:ldo :lfdoSame, on the location list
:cfilter {pat}Narrow qf to entries matching {pat}

Bulk refactor recipe: picker → filter → <C-q> → :cfdo %s/old/new/g | update

Leader groups

Lowercase = narrow/buffer; uppercase = broad/workspace. Group letter uses the case opposite to its direct shortcut: <Leader>b = buffer picker, <Leader>B* = buffer group.

<Leader>B+Buffer (alternate · delete · scratch · wipeout)
<Leader>p+Picker (catalog of mini.pick sources)
<Leader>g+Git (diff · log · commit · blame · overlay)
<Leader>c+Code (LSP · comment · notes)
<Leader>o+Session (write · read · delete)
<Leader>v+Visits (mini.visits frecency)
